YOLO

YOLO는 훌륭한 개발자들이 개발한 Tensorflow, Caffe, Darknet 등의 딥러닝 오픈소스 프레임워크이다. YOLO는 사물 인식에 많이 사용되었으며, 그중 우리는 차량과 차번을 인식하는데 필요한 기술이라 생각하여 사용하기로 하였습니다.

웹 서버

Flask 기반의 웹서버를 통한 접속

Flask

플라스크(Flask)는 파이썬으로 작성된 마이크로 웹 프레임워크의 하나로 웹서버를 제작하는데 사용이 되었습니다. 특히 웹 서버단에서 계정관리를 하기위해 세션처리를 해주는 경우, 또는 비 정상적 접속을 막기 위해 사용하였습니다.

Bootstrap 과 도식화 데이터

Bootstrap을 통해 비 반응형이 아닌 사용자와의 반응형 웹 템플릿 제작을 하였으며, GoogleChart, KakaoMap과 같은 정보 표시가 가능한 스크립트를 가져와서 인식된 차량의 정보를 종합적으로 최근 1주일 또는 금일 검거율 등을 그래프로 표시 및 검거 지역 비율을 도식화하여 보기 편하게 만들었습니다.

MySQL 데이터 베이스

데이터 베이스는 딥 러닝과 공통되는 부분으로서 딥 러닝 부분에서는 데이터인 차량과 차번을 인식하여 데이터 베이스에 넣으며, 넣은 데이터 베이스를 각종 쿼리문(조건문)을 통하여 원하는 값들을 추출하여 사용자에게 보기좋은 테이블 형태로 제공합니다.
